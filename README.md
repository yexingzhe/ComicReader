ComicReader
===========

a Comic Reader for windows 7 using register&amp;php Cli&amp;bat&HTML5

1.extract to your dir
2.edit line 7 in picplayer.reg set the path before run.bat and save it
3.import the picplayer.reg
4.right-click on the Comic dir to load it


采用了windows批处理技术、PHP Cli技术、windows注册表技术及HTML5技术

使用方法：
1、解压到一个固定目录
2、修改picplayer.reg第7行 run.bat前的路径
3、执行导入注册表
4、在你想要看漫画的目录上右键点击“用网页看图”
